# Outputs

This directory contains pre-generated result summaries from the submitted I-SEM arbitrage analysis.

## Files

| File | Description |
|---|---|
| `original_model_performance.csv` | Reported classification and walk-forward CV metrics by transition |
| `original_trading_performance.csv` | Reported strategy performance at confidence threshold 0.20 |
| `original_market_price_summary.csv` | Reported summary statistics for cleaned market prices |

The proprietary Energia raw dataset is not included. New outputs from a private reproduction run will be written here as `model_performance.csv`, `trading_results_all_thresholds.csv`, and `leaderboard.csv`.
